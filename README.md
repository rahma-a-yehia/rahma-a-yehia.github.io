# El Studio Website

A responsive, static website for El Studio — “From idea to ideal.”

## Preview locally

Open `index.html` directly in a browser, or run any static web server in this folder.

## Publish for free

The folder can be deployed unchanged to GitHub Pages, Cloudflare Pages, Netlify, or any static hosting provider.

## Contact actions

The project form prepares a structured email to `el.studio.for.design@gmail.com` and offers a WhatsApp alternative to `+20 114 903 7653`. It does not falsely report a message as sent.
